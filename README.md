# 🇺🇸 English

## Overview

Health Booking Flow is a frontend-focused UX/UI case study and responsive web application prototype designed to simplify the process of booking medical appointments.

The project explores how a healthcare experience can:

- Reduce cognitive load
- Minimize unnecessary steps
- Improve clarity and decision-making
- Create a faster and more human-centered booking flow

The application is being developed as a mobile-first responsive web app using React and Vite.

---

## Project Goals

The goal of this project is not to recreate a complete healthcare platform.

Instead, it focuses on a single core experience:

> Booking a medical appointment quickly and without friction.

The flow prioritizes:

- Clear hierarchy
- Fast decision-making
- Simple navigation
- Accessible interactions
- Consistent visual system

# 🇪🇸 Español

## Descripción

Health Booking Flow es un case study UX/UI orientado a frontend y un prototipo de aplicación web responsive diseñado para simplificar el proceso de reserva de citas médicas.

El proyecto explora cómo una experiencia digital en salud puede:

- Reducir la carga cognitiva
- Minimizar pasos innecesarios
- Mejorar la claridad y la toma de decisiones
- Crear un flujo más rápido y humano

La aplicación está siendo desarrollada como una web app responsive mobile-first utilizando React y Vite.

---

## Objetivos del proyecto

El objetivo de este proyecto no es recrear una plataforma médica completa.

En cambio, se centra en una experiencia concreta:
